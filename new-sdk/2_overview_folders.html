With the Redline release, all Crashday files and mods are now stored inside .cpk files. These files are just .zip archives and contain all of the game's content. If you go to the ...\Crashday\data folder you will find all of the game's .CPK content packages.

Do not edit the .cpk files inside ...\Crashday\data in any way. The game's anticheat will kick in and stop you from running the game if these files get tamperered with! If you want to change any content, create or use mods to do so.

.cdx files are encrypted packages and can not be opened.

This is a short overview of all the most modding-relevant folders:

ambience

Contains ambience definition files. These are used to change weather textures and change the mood (colors) of the ambience.

career

Contains all career missions definition files; including ladder.lst which contains career enemies' rankings

cars

Contains a subfolder for every car, contating different definition files

dbs

Contains the .dbs file, which contains all the main game and physics parameters

dynamics

Contains dynamic object definitions

editor

Contains track editor category sorting definitions

fonts

Contains all the fonts, including their sizes and conversion tables

loc

Contains localizations and the credits file

models

Contains all the models

models\cars

Every subfolder in this folder contains a car's model

music

Contains music and in-game soundtracks

particles

Contains particle definition files

shop

Contains career shop definition files

shop\cars

Contains the weapons shop list. Every subfolder has its corresponding car's shop list

sounds

Contains all the sounds

textures

Contains all the textures

tiles

Contains definition files for track pieces (tiles), which contains information such as tile dimensions, vegetation, AI pathing definitions and more

vegetat

Contains vegetation definition files

wheels

Contains wheel definition files
