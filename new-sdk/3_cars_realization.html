This section explains the handling and creation of vehicles in Crashday.


Each car exists as a unique subfolder in [crashday]\trkdata\cars and is automatically recognized by the game if it holds the file "carinfo.cca" inside (more on that later). Any files the car uses are contained in this one folder, except for any custom wheels (if any available at all).

To share your custom cars with others use the Car Manager tool to export the car folder into a .cpk archive (Crashday car package). To install custom cars, download the custom car's .cpk file and import it using with the Car Manager as well (see Car Manager).

If you installed or created a car correctly, it should appear available in the game's career shop.
