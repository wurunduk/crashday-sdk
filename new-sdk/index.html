Welcome to the Software Development Kit for Crashday: Redline Edition.
Since the Redline release, a lot of things regarding mod recreation are a little different. This new section of the SDK is meant to help you understand all the changes to Crashday when it comes to mod creation. Most of the old sdk should still be applicable, except some rare cases.

The new CD:RE SDK is provided and supported by the community. Feel free to contribute and ask for help in our discord channel, but please be nice.
